# Ghostwire — downloads & updates

Public distribution for **Ghostwire** (Notch Command Center). This repo holds **binaries only**:

- **Download the latest app:** [releases/latest](https://github.com/Ghostwire-Design/ghostwire-releases/releases/latest)
- **Sparkle update feed:** [`appcast.xml`](https://raw.githubusercontent.com/Ghostwire-Design/ghostwire-releases/main/appcast.xml)

Installed copies auto-update via Sparkle from the feed above.

## Source & license

Ghostwire is a GPL fork (via Buddi → boring.notch). The application source is maintained
privately; **corresponding source is available on request** to anyone who receives the binary,
in keeping with the GPL. Open an issue here to request it.
